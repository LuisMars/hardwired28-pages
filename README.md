# Hardwired 28

Corporate cablepunk skirmish wargame for 28mm miniatures.

Website: [hardwired28.luismars.com](https://hardwired28.luismars.com)

## About

Hardwired 28 is a tabletop skirmish game set in The Great Complex, a decaying vertical megastructure where wired connection is sacred and wireless is heresy. Players lead squads of contractors through missions against hostile factions.

## Contents

- **Core Rules** - Movement, combat, actions, and equipment
- **Hostiles** - Six hostile factions with unique behaviors
- **Campaign** - Linked missions, exploration, injuries, and promotions
- **Warband Builder** - Interactive squad creation tool
- **Scenario Generator** - Mission setup and deployment

Available in English and Spanish.
